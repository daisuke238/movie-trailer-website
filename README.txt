Entertainment Center Website

Steps to run the program (assumes Python/IDLE installed):

1. Place following files in the same folder:
   "entertainment_center.py"
   "fresh_tomatoes.py"
   "media.py"

2. Double click "entertainment_center.py"

If unsuccessful:

3. Run IDLE (Python GUI)

4. Open entertainment_center.py in IDLE

5. Go to Run -> Run Module (or press F5)
